O que é HTML, CSS e JavaScript?

HTML

HTML é a linguagem de marcação que define a estrutura do seu conteúdo web.

<article>
    <h1>Título</h1>
    <p>Conteúdo</p>
</article>

Use para: Estruturar conteúdo web, criar links, formulários e elementos semânticos.

CSS

CSS é a linguagem de estilização que controla a apresentação visual dos elementos HTML.

.card {
    color: #333;
    padding: 1rem;
    border-radius: 8px;
}

Use para: Estilizar elementos, criar layouts e adicionar animações.

JavaScript

JavaScript é a linguagem de programação que adiciona interatividade às páginas web.

button.addEventListener(
    'click', 
    () => alert('Olá!')
);

Use para: Criar interações, validar formulários e atualizar conteúdo dinamicamente.
